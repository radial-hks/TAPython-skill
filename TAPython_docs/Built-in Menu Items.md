## Built-in Menu Items[¶](#built-in-menu-items)

This page documents the built-in menu items provided by TAPython.

## TA Python Menu[¶](#ta-python-menu)

The TAPython plugin adds a "TA Python" menu to the Unreal Editor with the following default items:

### Python Console[¶](#python-console)

Opens the Python Console for interactive Python development.

### Chameleon Gallery[¶](#chameleon-gallery)

Opens the Chameleon Gallery - a collection of UI examples.

### Object Detail Viewer[¶](#object-detail-viewer)

A built-in tool for viewing and editing object properties.

## Additional Built-in Tools[¶](#additional-built-in-tools)

TAPython includes several built-in tools:

- **Chameleon Gallery**: UI component gallery
- **Object Detail Viewer**: Property editor
- **Shelf**: Asset organization tool

See [Built-in Tools](https://www.tacolor.xyz/tapython/built-in_tools.html) for more details.

## Customizing Menu[¶](#customizing-menu)

You can customize the menu through MenuConfig.json. See [Add Menu Items to Unreal Editor](https://www.tacolor.xyz/tapython/add_menu_items_to_unreal_editor.html) for details.
