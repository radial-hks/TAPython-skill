## TA Python ChameleonData API[¶](#ta-python-chameleondata-api)

This page provides an overview of the ChameleonData API.

## Overview[¶](#overview)

ChameleonData is the key communication channel between UI widgets and Python code. Python code can get and set widget content through it.

## Resources[¶](#resources)

All the ChameleonData API can be found in [ChameleonDataAPI](https://www.tacolor.xyz/pages/ChameleonDataAPI.html)
