## Chameleon Gallery[¶](#chameleon-gallery)

This page describes the Chameleon Gallery - a comprehensive UI component gallery in TAPython.

## Overview[¶](#overview)

The Chameleon Gallery is a built-in TAPython tool that showcases various UI components and layouts available in Chameleon.

## What You'll Find[¶](#what-youll-find)

The gallery includes examples of:

- **Basic Widgets**: Buttons, Text, Images
- **Layouts**: Vertical Box, Horizontal Box, Grid, Canvas
- **Advanced**: Scroll Boxes, Lists, Trees
- **Interactions**: Drag & Drop, Context Menus

## Using the Gallery[¶](#using-the-gallery)

1. Open the Gallery from **TA Python > Chameleon Gallery**
2. Browse through different UI patterns
3. Click on any component to see its JSON definition
4. Copy and adapt for your own tools

## Learning from Examples[¶](#learning-from-examples)

Each example in the gallery shows:
- Visual preview
- JSON structure
- Python callback examples
- Best practices
