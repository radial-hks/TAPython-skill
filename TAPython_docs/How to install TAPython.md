## How to install TAPython[¶](#how-to-install-tapython)

This guide explains how to install TAPython in Unreal Engine.

## Installation Steps[¶](#installation-steps)

1. **Download TAPython** from the GitHub repository or the Unreal Engine Marketplace

2. **Locate your Unreal Engine installation folder**
   - For UE4: `UE4_Version/Engine/Plugins/`
   - For UE5: `UE5_Version/Engine/Plugins/`

3. **Extract the TAPython plugin** to the Plugins folder
   - The folder structure should be: `Plugins/TAPython/`

4. **Restart Unreal Engine** or reload the plugin

5. **Verify installation**
   - Open the Python Console in Unreal Editor
   - Type `import unreal` to verify Python is working
   - You should see the TA Python menu in the main menu bar

## Required Unreal Engine Version[¶](#required-unreal-engine-version)

- UE 4.27 or later
- UE 5.0 or later

## Troubleshooting[¶](#troubleshooting)

If the plugin doesn't appear:

1. Check that the plugin folder is in the correct location
2. Verify the plugin files are not blocked (Windows)
3. Check the Output Log for any error messages
