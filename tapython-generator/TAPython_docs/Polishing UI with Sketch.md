## Polishing UI with Sketch[¶](#polishing-ui-with-sketch)

This guide explains how to polish your Chameleon UI using Sketch, a design tool.

## Overview[¶](#overview)

Sketch is a vector graphics editor for macOS. You can design your UI in Sketch and then export it for use with TAPython Chameleon.

## Workflow[¶](#workflow)

1. **Design in Sketch**: Create your UI mockups
2. **Export Assets**: Export images and SVGs
3. **Import to UE**: Add assets to your project
4. **Connect to Chameleon**: Reference assets in your JSON

## Tips for Sketch Design[¶](#tips-for-sketch-design)

- Use consistent spacing and padding
- Follow Unreal Editor style guidelines
- Test at different DPI scales
