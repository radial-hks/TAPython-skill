## Built-in Tools[¶](#built-in-tools)

This page documents the built-in tools provided by TAPython.

## Overview[¶](#overview)

TAPython comes with several pre-built tools to help you get started.

## Chameleon Gallery[¶](#chameleon-gallery)

A comprehensive gallery showcasing various UI components and layouts. Useful for:
- Learning UI patterns
- Finding examples for your tools
- Testing UI components

## Object Detail Viewer[¶](#object-detail-viewer)

A tool for viewing and editing properties of selected objects in the editor.

Features:
- View object properties
- Edit property values
- Filter and search properties

## Shelf[¶](#shelf)

An asset organization and quick-access tool.

Features:
- Drag and drop assets
- Quick access to frequently used assets
- Customizable categories

## Accessing Built-in Tools[¶](#accessing-built-in-tools)

Access built-in tools through:
- TA Python menu in the editor
- Python commands
- Custom menu configurations
